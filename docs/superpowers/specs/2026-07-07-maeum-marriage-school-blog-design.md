# 마음결혼학교 심리 콘텐츠 매거진 디자인 스펙

## 목표

`마음결혼학교`는 커플과 부부가 관계, 위기, 성심리에 관한 글을 오래 읽고 다시 찾아오게 만드는 심리 콘텐츠 매거진형 블로그다.

상담소처럼 보이는 정보 나열보다, 차분하고 깊이 있는 글을 읽게 만드는 편집 매체 느낌을 우선한다.

## 핵심 방향

- 상단 메뉴는 5개만 사용한다.
- 세부 주제는 메뉴에 노출하지 않고 글 태그로만 사용한다.
- 첫 화면은 브랜드 소개보다 글 탐색을 우선한다.
- 디자인은 따뜻하지만 과하게 감성적이지 않게 만든다.
- 민감한 주제도 신뢰감 있게 보이도록 여백, 타이포그래피, 색 대비를 안정적으로 둔다.

## 상단 메뉴

1. 홈
2. 관계심리
3. 위기와 회복
4. 성심리
5. 전체 글

## 페이지 구조

### 홈

홈은 방문자가 바로 글을 읽기 시작하는 입구다.

구성:

- 브랜드명 `마음결혼학교`
- 짧은 소개 문장
- 대표/추천 글 1개
- 최신 글 목록
- 많이 읽은 글 목록
- 세 카테고리로 이동하는 간단한 영역

### 관계심리

연애, 결혼, 관계자존감, 성격차이, 대화, 갈등 등 관계 전반을 다루는 카테고리 페이지다.

화면에는 카테고리 제목, 짧은 설명, 글 카드 목록만 둔다.

### 위기와 회복

외도, 이혼위기, 신뢰 붕괴, 재결합, 관계 회복처럼 깊고 무거운 주제를 다루는 카테고리 페이지다.

톤은 자극적 제목보다 안정감 있는 제목과 요약을 우선한다.

### 성심리

커플 성생활, 심인성 성기능, 성중독, 성문제 관련 글을 다루는 카테고리 페이지다.

민감한 주제이므로 과장된 표현이나 선정적 이미지는 쓰지 않는다.

### 전체 글

모든 글을 최신순으로 볼 수 있는 아카이브 페이지다.

초기 버전에서는 글 목록만 제공하고, 검색/정렬/태그 필터는 이후 확장 기능으로 둔다.

## 디자인 톤

참고 방향은 `melyssagriffin.com`의 개인 브랜드형 매거진 감각이다. 그대로 복제하지 않고 한국 심리 콘텐츠에 맞게 더 차분하게 정리한다.

- 배경: 따뜻한 오프화이트
- 본문 텍스트: 짙은 먹색
- 포인트 색: 깊은 녹색 또는 차분한 버건디 계열 중 하나
- 제목: 세리프 또는 세리프 느낌의 고급스러운 한글 폰트
- 본문: 읽기 좋은 산세리프 한글 폰트
- 카드: 얇은 테두리, 작은 radius, 충분한 여백
- 이미지: 인물 클로즈업보다 손, 공간, 책상, 빛, 창가처럼 은유적이고 차분한 이미지

## 콘텐츠 모델

글은 다음 필드를 가진다.

- 제목
- 요약
- 카테고리
- 태그
- 작성일
- 읽는 시간
- 대표 이미지
- 본문

메뉴에 넣지 않을 태그 예시:

- 외도
- 이혼위기
- 성격차이
- 관계자존감
- 커플 성생활
- 심인성 성기능
- 성중독
- 성도착
- 성폭력 가해자 상담
- 아동청소년 성문제

## 초기 구현 범위

초기 버전은 정적 블로그 구조로 만든다.

- 홈
- 관계심리 카테고리
- 위기와 회복 카테고리
- 성심리 카테고리
- 전체 글
- 샘플 글 데이터
- 반응형 상단 네비게이션

GitHub 업로드와 Vercel 배포를 고려해 Next.js 또는 Vite 기반 정적 프론트엔드로 구성한다.

## 성공 기준

- 상단 메뉴가 5개만 보인다.
- 세부 주제는 메뉴에 보이지 않는다.
- 첫 화면에서 글을 바로 읽고 싶게 만든다.
- 모바일에서도 메뉴와 글 목록이 단순하게 보인다.
- 이후 실제 글 데이터로 교체하기 쉽다.
